Supabase 연동 ⚙️
설정하면 수업 데이터가 클라우드에 저장되어 수강생과 공유됩니다
📖 설정 방법 (처음 한 번만)
  1. supabase.com 에서 무료 계정 만들기
  2. 새 프로젝트 생성 (이름 자유, 비밀번호 기억해두기)
  3. 좌측 메뉴 SQL Editor 클릭 → 아래 SQL 실행:
create table if not exists catalog ( id text primary key, data jsonb not null, updated_at timestamptz default now() ); create table if not exists notices ( catalog_id text primary key, content text, updated_at timestamptz default now() ); alter table catalog enable row level security; alter table notices enable row level security; create policy "public read catalog" on catalog for select using (true); create policy "public write catalog" on catalog for all using (true); create policy "public read notices" on notices for select using (true); create policy "public write notices" on notices for all using (true);
  1. 좌측 메뉴 Settings → API 클릭
  2. Project URL과 anon public key 복사
  3. 아래에 붙여넣기 후 저장
🔗 연결 정보 입력
🔄 데이터 동기화

연결 후 아래 버튼으로 현재 수업 데이터를 클라우드에 업로드하세요. 수강생들은 앱을 열 때 자동으로 최신 수업 목록을 불러옵니다.
